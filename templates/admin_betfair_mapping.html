How to Map Betfair Markets
  1. Go to Betfair Exchange
  2. Navigate to the race you want to map
  3. Copy the Market ID from the URL (the number after /market/)
  4. For each horse, you can find their Selection ID in the page source or API
  5. Paste the IDs into the mapping form above and save
Tip: If you only set the Market ID, the system will try to match horses by name automatically.
